# Creating Parameter Studies with Strategy Expansion

## Overview

Strategy expansion (new in 0.7.0) allows you to automatically create multiple analyses from parameter matrices. This is powerful for:
- Systematic comparison studies
- Exploring the impact of different settings
- Waveform model comparisons
- Prior sensitivity studies

## Key Concepts

- Parameter matrices
- Strategy templates
- Automatic analysis generation
- Result comparison

## Example Use Cases

1. Compare multiple waveform approximants on the same event
2. Test sensitivity to different priors
3. Evaluate different sampler configurations
4. Multi-pipeline comparisons
