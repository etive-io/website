# Writing and Validating Analysis Blueprints

## Overview

Blueprints are YAML configuration files that define events, analyses, and defaults for asimov projects. asimov 0.7.0 introduces:
- Blueprint validation with pydantic
- Pipeline-specific prior interfaces
- Early error detection
- Better error messages

- Blueprint structure
- Event specifications
- Analysis configurations
- Prior definitions
- Validation rules
