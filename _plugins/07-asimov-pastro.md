# asimov-pastro

Integration between Asimov and Pastro for astrophysical parameter inference from gravitational wave posteriors.

## Features

- **Population analysis** from single-event posteriors
- **Astrophysical parameter inference** beyond detector frame
- **Bayesian hierarchical models** for population studies
- **Integration with asimov workflows**

## Installation

```bash
pip install asimov-pastro
```

## Purpose

Pastro takes parameter estimation results and infers astrophysical population properties, such as merger rates, mass distributions, and other source characteristics that inform our understanding of compact object populations.
