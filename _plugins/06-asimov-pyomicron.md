# asimov-pyomicron

Integration between Asimov and Pyomicron for automated glitch detection in gravitational wave data.

## Features

- **Automated glitch detection** using omicron algorithm
- **Integration with data quality assessment**
- **Workflow integration** for monitoring and flagging
- **Customizable trigger parameters**

## Installation

```bash
pip install asimov-pyomicron
```

## Purpose

Glitches are transient noise artifacts in gravitational wave detector data. Pyomicron automatically identifies these using excess power clustering, which is important for understanding data quality and validating gravitational wave candidates.
